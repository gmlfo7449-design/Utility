색상 코드 변환 가이드

웹 디자인과 개발에서 색상은 HEX, RGB, HSL, CMYK, HSV 등 다양한 포맷으로 표현됩니다. CSS에서는 주로 HEX(#FF0000)와 RGB(rgb(255,0,0))를 사용하고, 디자인 도구에서는 HSL이나 HSV를, 인쇄물에서는 CMYK를 사용합니다. 이 도구를 사용하면 어떤 포맷의 색상 코드든 입력하는 즉시 나머지 모든 포맷으로 자동 변환됩니다.

색상 피커에서 직접 색상을 선택하거나 Hue(색상) 슬라이더로 기본 색상을 바꿀 수 있으며, Alpha(투명도) 슬라이더로 반투명 색상도 만들 수 있습니다. 모든 변환은 브라우저에서 실시간으로 처리되며, 변환된 코드는 복사 버튼 한 번으로 클립보드에 저장할 수 있습니다.

자주 묻는 질문 (FAQ)

색상 코드를 직접 입력할 수 있나요?
네, 모든 입력 필드에서 직접 색상 코드를 입력할 수 있습니다. 예를 들어 RGB 필드에 rgb(255, 0, 0)을 입력하면 빨간색으로 변환되고, HEX 필드에 #FF0000을 입력해도 동일합니다. HSL, CMYK, HSV 등 어떤 포맷이든 입력하면 나머지 포맷이 자동으로 변환됩니다.
HEX 색상 코드란 무엇인가요?
HEX 코드는 # 기호와 6자리 16진수로 색상을 표현하는 방법입니다. 예를 들어 #FF0000은 빨강, #00FF00은 초록, #0000FF는 파랑입니다. 앞 두 자리는 Red, 중간 두 자리는 Green, 마지막 두 자리는 Blue 값을 나타냅니다.
RGB와 RGBA의 차이점은 무엇인가요?
RGB는 Red, Green, Blue 세 가지 값(0~255)으로 색상을 표현합니다. RGBA는 여기에 Alpha(투명도, 0~1) 값이 추가된 것입니다. Alpha가 1이면 불투명, 0이면 완전 투명입니다.
CMYK는 언제 사용하나요?
CMYK(Cyan, Magenta, Yellow, Key/Black)는 인쇄물에서 사용하는 색상 모델입니다. 모니터는 빛을 섞어 색을 만들지만(RGB), 프린터는 잉크를 섞어 색을 만들기 때문에 인쇄용 디자인에서는 CMYK로 변환하는 것이 중요합니다.
HSL과 HSV의 차이는 무엇인가요?
HSL은 Hue(색상), Saturation(채도), Lightness(밝기)로, HSV는 Hue(색상), Saturation(채도), Value(명도)로 색상을 표현합니다. HSL에서 Lightness 50%가 가장 순수한 색상이고, HSV에서 Value 100%가 가장 밝은 색상입니다.
