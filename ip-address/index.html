자주 묻는 질문 (FAQ)

IP 주소란 무엇인가요?
IP 주소(Internet Protocol Address)는 인터넷에 연결된 기기에 부여되는 고유한 번호입니다. 웹사이트에 접속하거나 데이터를 주고받을 때 이 주소를 통해 기기를 식별합니다. 공인 IP는 외부에서 보이는 주소이고, 사설 IP는 가정이나 회사 내부 네트워크에서 사용하는 주소입니다.
IPv4와 IPv6의 차이는 무엇인가요?
IPv4는 192.168.0.1처럼 숫자 4개로 구성된 주소 체계이고, IPv6는 2001:0db8:85a3::8a2e 같은 더 긴 형태의 주소 체계입니다. IPv4 주소가 부족해지면서 IPv6가 점차 보급되고 있습니다.
내 IP 주소가 바뀔 수 있나요?
네. 대부분의 가정용 인터넷은 유동 IP를 사용하므로 공유기를 재시작하거나 일정 시간이 지나면 IP가 변경될 수 있습니다. 고정 IP는 별도로 신청해야 하며 주로 서버 운영 등에 사용됩니다.
IP 주소로 정확한 위치를 알 수 있나요?
IP 기반 위치 정보는 대략적인 지역(도시 수준)까지만 파악 가능합니다. 정확한 집 주소나 건물 위치를 알 수는 없습니다. 표시되는 위치는 ISP의 서버 위치일 수도 있어 실제 위치와 차이가 있을 수 있습니다.
VPN을 사용하면 다른 IP가 보이나요?
네. VPN을 사용하면 VPN 서버의 IP 주소가 표시됩니다. VPN이 정상적으로 작동하는지 확인하려면 이 도구로 IP를 조회해서 VPN 서버 위치의 IP가 표시되는지 확인하면 됩니다.
